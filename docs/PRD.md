# PRD：元宵节短平快项目 —— 烟台专属 AI 猜灯谜与定制海报引流

## 1. 项目背景
元宵节流量窗口短（2-3天），相比囤货类实体玩法，内容引流+本地商家广告合作风险更低、上线更快。项目目标是在3天内上线可用工具，批量生成“烟台本地化灯谜内容 + 海报生成提示词”，用于小红书/抖音同城分发。

## 2. 目标与成功指标
### 2.1 核心目标
- 快速生成可发布内容：烟台本地灯谜、答案、互动话术。
- 输出可直接喂给绘图模型的海报提示词（赛博朋克/新中式）。
- 帮助运营侧完成“流量 → 商家赞助/私域沉淀”闭环。

### 2.2 指标（MVP）
- 生成速度：单次 20 条灯谜 ≤ 10 秒。
- 可用性：移动端可用，主要操作不超过 3 步。
- 内容产出：支持地标/特产/方言三个主题维度。
- 运营转化：每条内容自带“引导评论/私信/到店活动”文案模板。

## 3. 用户与场景
### 3.1 目标用户
- 本地自媒体运营者
- 烟台本地商家（餐饮、剧本杀、活动主办方）
- 做同城私域的个体创业者

### 3.2 核心场景
1. 运营输入主题与数量，批量生成灯谜。
2. 运营选择视觉风格，生成海报提示词并复制到AI绘图工具。
3. 一键复制“评论区置顶广告文案/私域引流文案”。

## 4. 功能范围（3天上线MVP）
### 4.1 必做
- 灯谜生成器（本地化词库 + AI语气模板）。
- 海报提示词生成器（赛博朋克/新中式）。
- 发布文案模板（抖音/小红书风格）。
- 简单Web页面（单页操作）。

### 4.2 暂不做
- 用户登录与权限系统。
- 自动发布到平台API。
- 商家CRM与支付系统。

## 5. 功能需求
### 5.1 灯谜生成
- 输入：主题（地标/特产/方言/混合）、数量（5-50）、语气（趣味/诗意）。
- 输出：`谜面 + 谜底 + 解释 + 互动引导句`。
- 内容规则：至少70%包含烟台本地元素。

### 5.2 海报提示词
- 输入：风格（赛博朋克/新中式）、主视觉（灯笼/海岸线/地标）、色调。
- 输出：
  - 绘图模型提示词（中文+英文关键词）
  - 标题与副标题
  - CTA（评论区互动/私信领答案）

### 5.3 转化文案
- 商家合作模板：猜中到店折扣。
- 私域引导模板：加微信领答案/红包。

## 6. 非功能需求
- 页面首屏加载 < 2 秒（本地部署环境）。
- API响应结构统一JSON。
- 对非法参数返回清晰报错。

## 7. 技术方案（MVP）
- 后端：Python `http.server`（零外部依赖，快速上线）
- 前端：原生HTML + CSS + JS
- 生成策略：
  - 第一阶段：本地模板+词库生成，确保离线可用。
  - 第二阶段：可切换接入大模型API（预留接口）。

## 8. 里程碑（3天）
- Day 1：PRD + 页面原型 + 核心数据结构。
- Day 2：后端API与前端联调。
- Day 3：内容调优、测试、上线与素材产出。

## 9. 风险与应对
- 风险：内容同质化。
  - 应对：加入随机语气模板与本地词库扩展。
- 风险：节日窗口短。
  - 应对：优先发布高互动版本（20条谜面+1张海报）。
